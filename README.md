# React Router Redux Template

## Getting Started


### Build

```
yarn install
```

```
yarn run build:front:dev
```

```
open http://localhost:8888
```

### Spec

```
yarn run test:front
```


## Libraries

* React (v16)
* Redux (v3)
* redux-saga
* react-router (v4)
* styled-components (v2)
